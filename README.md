## **浪潮LSI-2308 刷直通(IT)或阵列(IR)**

### **注意事项**
	1.只能efi刷固件
	2.避免电脑插入多张LSI-2308卡
	3.执行命令过程中避免断电

### 1.需要准备的工具


| 工具 | 备注 |
| ------- | ------- |
|   U盘      |   1G以上，格式化成FAT32格式      |
|   resource.zip      |   刷新固件所需要的相关文件      |

### 2.刷固件

第一步: 把压缩包(resource.zip)解压到u盘根目录

第二步: 重启电脑选择该usb efi启动

第三步: 进入efi命令行依次执行以下命令
### 
	输入命令map找到usb所在驱动器，一般fs0就是当前u盘，
	如不是fs0则输入对应名称加 : (如fs1:)进入U盘目录。
**此处注意，不要漏掉“:”，后面所有命令中涉及的符号，记得都要用英文半角。

### 
	# 查看阵列卡
	sas2flash.efi -listall

### 
	# -o 启用高级命令模式 -e 删除固件的指定区域
	sas2flash.efi -o -e 6
	
### 
	# -f 固件更新 -b BIOS更新
	sas2flash.efi -o -f 2118it.bin -b mptsas2.rom


### 3.验证
	重启看见阵列卡页面根据提示按ctrl+c 查看是否为it模式
